Requirements

System Requirements:

  • Windows All
  • Linux
  • Mac OS X



Application Requirements:

  • Java JDK 1.7 (https://jdk7.java.net/download.html)


If the program takes too long to start or gives an exception error, check Java version used on launch via the JAVA_HOME and PATH environment variables and make sure they're pointing on the jdk 1.7 bin folder.



Package Requirements:


Here are the requirements for the resulting setup packages you build:


  • with IzPack: depends on Java JRE 6 and newer
  • with NuGet: depends on Chocolatey
