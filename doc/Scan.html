1 - Scan the directory containing the files you want to include in your package


Press the button to choose a directory containing all of the files you need in your installer, then filter some of them if you need so. A 'simple scan' treats folders as single packs, and 'recursive scan' gets packs from inside directories until 'depth' is reached.
